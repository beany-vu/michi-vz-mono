# @michi-vz

Framework-agnostic data-visualization library: a plain-TS rendering **engine** (`@michi-vz/core`),
native **web components** (`@michi-vz/wc`), thin wrappers for **React / Vue / Angular / Svelte**,
and an opt-in client-side **AI insights** layer (`@michi-vz/insights`).

Monorepo: pnpm workspaces + Turborepo + Changesets + tsup. See the plan for architecture details.
